Bootstrap-outline-btn

Styles for bootstrap buttons that are outlined. Also includes styles for inverses of the buttons.

View the Project on GitHub pdsullivan/Bootstrap-Outline-Btn

Here are some examples of the buttons with the code as well

See, they are the same as the bootstrap 3 buttons that you are used to use.





The code for these examples.

<button class="btn btn-success btn-outline">Success Outline Button</button> <button class="btn btn-primary btn-outline">Primary Outline Button</button> <button class="btn btn-danger btn-lg btn-outline">Large Danger Outline Button</button>

Inverse Examples

These work the same way but they are the inverse of the above buttons. Just add a "-inverse" onto the end of the class and boom.





The code for these examples.

<button class="btn btn-success btn-outline-inverse">Success Outline Inverse Button</button> <button class="btn btn-primary btn-outline-inverse">Primary Outline Inverse Button</button> <button class="btn btn-danger btn-lg btn-outline-inverse">Large Danger Inverse Outline Button</button>

Here is what the code looks like.
